# Project: Get Into Concert Photography

**Connected area:** Photography
**Status:** Active
**Target:** Shoot concerts professionally or semi-professionally
**Timeline:** By December 2026
**Why:** I love music, I love photography — concert photography is where both collide

---

## The Path (4 Phases)

### Phase 1: Foundation (Feb–Apr 2026)
Learn the technical skills needed for low-light, fast-action environments.

- [ ] Master manual mode (aperture, shutter speed, ISO triangle)
- [ ] Learn low-light photography techniques
- [ ] Study composition for dynamic subjects
- [ ] Practice burst/continuous shooting modes
- [ ] Edit 50+ photos in Lightroom/Capture One
- [ ] Study 5 concert photographers (style analysis)

**Time:** 3–4 hours/week (theory + practice)

### Phase 2: Practice & Portfolio (May–Jul 2026)
Build skills through real-world practice at accessible events.

- [ ] Shoot 5+ local events (open mics, street performances, DJ sets)
- [ ] Practice editing workflow — develop a consistent style
- [ ] Create a portfolio (Instagram or personal site) with 20+ best shots
- [ ] Learn music venue etiquette and photographer positioning
- [ ] Connect with 3+ local photographers or communities

**Time:** 4–5 hours/week + 2–3 events/month

### Phase 3: Real Gigs (Aug–Oct 2026)
Start shooting at actual concerts and build reputation.

- [ ] Attend 3+ small concerts with camera (where allowed)
- [ ] Reach out to 5 local venues/promoters for photo pass
- [ ] Build relationships with 2+ local artists/bands
- [ ] Get featured on an event page or artist social media
- [ ] Portfolio at 50+ curated shots

**Time:** 5–6 hours/week + 3–4 events/month

### Phase 4: Go Pro (Nov–Dec 2026)
Transition from hobbyist to recognized concert photographer.

- [ ] Get a press/photo pass for a mid-size concert
- [ ] Have work published (blog, magazine, artist page)
- [ ] Build a network of 10+ industry contacts
- [ ] Define your niche/style (intimate portraits? crowd energy? stage lighting?)
- [ ] Optional: First paid gig

**Time:** 6+ hours/week

## Gear Checklist
- [ ] Camera body (fast autofocus, good high-ISO performance)
- [ ] Fast lens: 50mm f/1.8 or 35mm f/1.4
- [ ] Zoom lens: 70-200mm f/2.8 (for later phases)
- [ ] Extra batteries + memory cards
- [ ] Editing software (Lightroom/Capture One)
- [ ] Portfolio website or dedicated Instagram
